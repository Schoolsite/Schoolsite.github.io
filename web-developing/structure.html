Содержание и структура сайта

Прежде, чем выбирать инструмент для создания сайта, необходимо определиться с его структурой и содержанием.


Для примера попробуем разработать структуру сайта для вашего класса. Для удобства учеников, родителей и учителей можно создать сайт школьного класса и делиться там необходимой информацией. Так она будет доступна для всех, наглядна и удобна в использовании, при условии, что сайт вы будете постоянно обновлять и дополнять.


Для начала нужно подумать, какая информация будет нужна пользователям сайта.


Для учеников — это может быть расписание уроков, домашние задания, конспекты уроков, ссылки от учителей на полезные ресурсы, информация о предстоящих мероприятиях и т. д. Для родителей: информация о родительских собраниях, важная информация от классного руководителя, советы от школьного психолога и т. п. Для учителей ваш сайт будет удобной площадкой для коммуникации с учениками, родителями.
